# CSCN8010 Practice Repository

## Project Description
This repository is created for practicing basic GitHub workflows and Python programming concepts for the CSCN8010 course.

The project demonstrates:
- Creating and managing a public GitHub repository
- Using Visual Studio Code with Git
- Writing basic Object-Oriented Programming (OOP) code in Python
- Documenting code using Markdown

## Files Overview
- `main.py`  
  A simple Python program that demonstrates Object-Oriented Programming using a `Student` class.

- `README.md`  
  Project documentation written in Markdown.

## How to Run
1. Open the project folder in Visual Studio Code.
2. Make sure Python is installed on your system.
3. Run the following command in the terminal:

```bash
python main.py
